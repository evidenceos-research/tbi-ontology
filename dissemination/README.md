# Dissemination Assets

This folder contains publication-ready communication drafts for the DAI -> TAPVI terminology update.

## Files

- `substack_dai_to_tapvi.md` - long-form educational article draft
- `linkedin_thread_dai_to_tapvi.md` - 10-post social thread draft

## Core message

- DAI is deprecated as an umbrella label in this ontology context.
- TAPVI (Traumatic Axonal and/or Punctate Cerebrovascular Injury) provides more accurate phenotype language.
- Structured terminology improves interoperability, reproducibility, and implementation quality.
